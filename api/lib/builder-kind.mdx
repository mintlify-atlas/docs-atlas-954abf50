## Overview

The `BuilderKind` enum represents the different MEV builder services supported by the library. Each variant corresponds to a specific builder with its own endpoint configuration.

## Type definition

```rust
#[derive(Clone, EnumIter, Display)]
pub enum BuilderKind {
    Titan,
    Beaver,
    Rsync,
}
```

## Variants

### Titan

Titan Builder endpoint configuration.

<ParamField path="endpoint" type="BuilderEndpoint">
  - **URL**: `https://rpc.titanbuilder.xyz`
  - **RPC Method**: `eth_sendPrivateTransaction`
</ParamField>

### Beaver

Beaver Build endpoint configuration.

<ParamField path="endpoint" type="BuilderEndpoint">
  - **URL**: `https://mevshare-rpc.beaverbuild.org`
  - **RPC Method**: `eth_sendPrivateRawTransaction`
</ParamField>

### Rsync

Rsync Builder endpoint configuration.

<ParamField path="endpoint" type="BuilderEndpoint">
  - **URL**: `https://rsync-builder.xyz`
  - **RPC Method**: `eth_sendPrivateRawTransaction`
</ParamField>

## Methods

### endpoint

Returns the endpoint configuration for the builder variant.

```rust
fn endpoint(&self) -> BuilderEndpoint
```

<ResponseField name="BuilderEndpoint" type="BuilderEndpoint">
  A struct containing the URL and RPC method name for the builder
</ResponseField>

<Note>
  This method is private and used internally by the `builder()` method.
</Note>

### builder

Creates a new `Builder` instance configured for this builder type.

```rust
pub fn builder(&self) -> Result<Builder, BuilderError>
```

<ResponseField name="Result" type="Result<Builder, BuilderError>">
  Returns a configured `Builder` instance, or `BuilderError::ClientError` if the HTTP client cannot be initialized
</ResponseField>

#### Implementation details

The method:
1. Retrieves the endpoint configuration via `endpoint()`
2. Builds a new HTTP client using `reqwest::Client::builder()`
3. Returns a `Builder` struct with the client, endpoint, and builder kind

    let rsync_builder = BuilderKind::Rsync.builder()?;
